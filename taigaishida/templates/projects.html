Crypto Tracking Bot For Slack

A Slack bot server that adds commands to users in a chat room.
Incorporates a python script that fetches crpyto information from coinmarketcap’s API and returns it to users.
Implemented a modular design by separating the bot functions from the main functions.
Also improved runtime by storing the coin ID’s in a hashmap on startup.

Embedded Space Invaders Clone

Space Invaders Clone written in assembly and flashed onto a LCP-2138 Arm Microcontroller using Keil5.
Made use of the GPIO and UART to receive and transmit data.
For example, I used the 7-segment-display to show the score and the LEDs to show life stock.
The game required designing and implementing algorithms for movement, attacking, collision detection and much more.
